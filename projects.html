Automated GIS Pipeline Processing Tool

Origin: IT 338 Final Project
Work in Progress: Refactor and Enhance for CS 499 Capstone

This tool was originally developed to automate pipeline data processing using ArcGIS Pro and ArcPy. As part of the CS 499 capstone enhancements, it will be refactored to include a command-line interface and eliminate reliance on proprietary software, transitioning to open-source libraries like GeoPandas and Shapely. Enhancements include:

  • Open-source transition for broader accessibility
  • CLI support for flexible, scriptable usage
  • Optimized buffer and spatial merge logic
  • MongoDB integration for scalable spatial data storage
  • Enhanced multiprocessing, logging, and error handling
